ScheduleEventHandler

The event handler callback functions for events triggered by the scheduler.
Type:
  • object
Properties:
Name Type Attributes Description
onQueue function <optional>
 triggered when a step has been queued up for playback.
onAnim function <optional>
 triggered when the playback position meets a queued step (used for animating steps).
onTween function <optional>
 triggered every render frame with the normalized playback position within the bar in the [0,1] range.

StepStamp

The position stamp of a step on the timeline (all values are 0-based).
Position stamps should only be created and modified with the appropriate step stamp methods (create, clone, offset) and never directly.
Type:
  • object
Properties:
Name Type Description
bar number the bar position of this step (the pattern is one bar on length).
beat number the beat position of this step.
step number the step position of this step (in the beat).
patternPos number the step position of this step (in the pattern).
guid number the unique absolute position of this step in playback.
Source:
  • mcad.pattern.scheduler.v1-0.js, line 7
Example

A pattern of 4 beats of 1/16th notes

// Values are 0-based (0 = 1st, 1 = 2nd, etc.)
myStepStamp = someScheduler.createStepStamp(1,3,0); // 2nd bar, 4th beat, 1st step, patternPos = 12, guid = 28

TimeStamp

The time stamp of a step on the timeline.
Time stamps should only be created and modified with the appropriate time stamp methods (create, clone) and never created or copied directly.
Type:
  • object
Properties:
Name Type Description
straight number the absolute straight time position of a note.
swing number the absolute swung time position of a note.
