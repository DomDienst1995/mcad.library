BufferLoader

An asynchronous buffer loader capable of loading multiple individual sample buffers and buffer lists simultaneously.

Constructs a buffer loader object.

Example

Load a single sample into a buffer

// Create web audio context
var audioCtx = new AudioContext();

// Create a buffer loader object
var bufferLoader = new BufferLoader(audioCtx); 

// URL of sample to load
var kickURL = "samples/909kick.wav"

// Load the kick sample
bufferLoader.loadBuffer(kickURL, function(result, url){
                
    // Sample failed to load/decode
    if(result === null) {
        console.log("Single file failed to load " + url);
    }
    // Sample was successfully loaded
    else {
    
        // Play back the kick sample
        var source = audioCtx.createBufferSource();
        source.buffer = result; 
        source.connect(audioCtx.destination);
        source.start();  
    }
});

loadBufferList(bufferList, urlList, onFinishednullable, onLoadnullable)

Loads a sequential array or key/value array of samples into a sequential or key/value array of buffers.
Parameters:
Name Type Attributes Description
bufferList Array.<AudioBuffer> | Array.<key, AudioBuffer> the destination array (sequential or key/value) the sample buffers will be loaded into.
urlList Array.<string> | Array.<key, string> the url array (sequential or key/value) dictating the location of the samples to be loaded.
onFinished function <nullable>
 the event handler to be called when all of the sample buffers requested have been loaded (or attempted).
onLoad function <nullable>
 the event handler to be called when a sample buffer in the array is loaded (or attempted).
Properties
Name Type Description
result object contains the result (key/value pairs) of a given sample buffer's loading/decoding:
Properties
Name Type Description
url string the url of the sample buffer that was loaded (or attempted).
key number | string the index (for sequential) or key (for key/value) of the sample buffer that was loaded (or attempted).
success boolean true if loading/decoding of this buffer was successful, false if not.

Examples

Loading an array of key/value samples

// Create web audio context
var audioCtx = new AudioContext();

// Create a buffer loader object
var bufferLoader = new BufferLoader(audioCtx); 

// List to hold all of our audio buffers
var sampleBuffers = {};

// Flag to determine whether or not the samples have finished loading
var samplesLoaded = false;
 
// Load a kick, hat and snare sample into the list and associate them with the "kick", "hat" and "snare" keys
bufferLoader.loadBufferList(sampleBuffers, {"kick": "samples/909kick.wav", "hat": "samples/909hat.wav", "snare": "samples/909snare.wav"}, function() {
    
    console.log("Finished loading sampleBuffers!");
    samplesLoaded = true;
});


// When the element with id "kick" is clicked...
$("#kick").click(function() {

    if(samplesLoaded) {

        // Play back the kick sample
        var source = audioCtx.createBufferSource();
        source.buffer = sampleBuffers["kick"]; // Or source.buffer = sampleBuffers.kick;
        source.connect(audioCtx.destination);
        source.start();  

    }
    else console.log("Sampled not loaded yet!");

});
