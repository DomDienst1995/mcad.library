Rotary

Rotary

A rotary widget that supports mouse and touch input (pinch and pan). The widget will handle all input and rotary mechanics and inform the user of value changes via the onChange event handler.

Constructor

new Rotary(selectornon-null, options)

Constructs a rotary widget object.
Parameters:
Name Type Description
selector string jQuery selector string to select the element that represents this widget.
options object config options to override defaults.
Properties
Name Type Attributes Default Description
image_src string path to the image file to be used for the rotary.
onChange number event handler for parameter value changes).
width number width of the widget (suffix with units, e.g. px or %).
height number width of the widget (suffix with units, e.g. px or %).
paramMin number minimum parameter value for this widget.
paramMax number maximum parameter value for this widget.
degMin number <optional>
 225 minimum degree position for this widget.
degMax number <optional>
 135 maximum degree position for this widget.
pinch number <optional>
 true allow pinch gestures to rotate widget.
pan number <optional>
 true allow pan gestures to rotate widget.
Source:
  • mcad.rotary.v1-0.js, line 45
Examples

Pitch knob with 300hz to 20khz parameter range

<div id="pitchKnob"></div>

<script>
   function pitchKnobChange(pValue, nValue) {
       console.log("Parameter value: " + pValue + "hz, normalized value: " + nValue); 
   }

   var pitchKnob = new Rotary("#pitchKnob", {image_src: "resources/knob.png", onChange: pitchKnobChange, width: "128px", height: "128px", paramMin: 300, paramMax: 20000});
</script>

Pitch knob with 300hz to 20khz parameter range but no pinch gestures

<div id="pitchKnob"></div>

<script>
   function pitchKnobChange(pValue, nValue) {
       console.log("Parameter value: " + pValue + "hz, normalized value: " + nValue); 
   }

   var pitchKnob = new Rotary("#pitchKnob", {image_src: "resources/knob.png", onChange: pitchKnobChange, width: "128px", height: "128px", paramMin: 300, paramMax: 20000, pinch: false});
</script>
